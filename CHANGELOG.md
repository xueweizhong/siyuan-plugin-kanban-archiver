- 报表内容提取增强 values.content 兜底与 contentValueObj 的 blockId 识别
- 报表内容提取支持 itemID → 绑定块ID 映射后回填内容
- 修复报表生成中 resolved 未定义导致报错的问题
- itemID → blockID 映射解析增强，修复内容仍为空的情况
- 报表内容提取增加样例内容与映射调试日志
- 报表调试日志增加 values 与 columns 输出，便于定位内容来源
